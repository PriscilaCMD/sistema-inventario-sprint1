# Sistema de Inventario – Sprint 1

Proyecto desarrollado en React como parte del seguimiento de Sprints, aplicando la metodología SCRUM en la planificación, desarrollo y mejora continua del sistema de inventarios.

## Objetivo del Sprint 1

Implementar el control básico de inventario permitiendo:

- Registrar productos
- Definir stock mínimo y máximo
- Registrar entradas de productos
- Actualizar el stock dinámicamente

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- Node.js

## Funcionalidad

La aplicación permite:

1. Registrar nuevos productos con stock inicial.
2. Definir niveles mínimo y máximo de inventario.
3. Registrar entradas de productos existentes.
4. Actualizar automáticamente el stock en pantalla.

Este incremento corresponde a las historias de usuario:
- PB2 – Registrar productos
- PB3 – Definir stock mínimo y máximo
- PB4 – Registrar entradas
